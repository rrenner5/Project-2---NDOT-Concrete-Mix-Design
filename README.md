# Project-2---NDOT-Concrete-Mix-Design
The repository contains files and information on the CIVE 202 Project #2 - Concrete Mix Design. This code automates the process of concrete mix design for NDOT. It takes formulas from Excel and converts in into a Python environment with user inputs and final mix design output tables.

The goal of this project is to automate the Nebraska Department of Transportation (NDOT) concrete mix design process by translating an existing concrete mix design model from an Excel-based workflow into a Python-based computational tool. The automated tool will allow users to enter project information, such as project number, control number, and material properties and mix design parameters, including class of concrete, target amounts, and specific gravity, in a structured, sequential manner, and will output a final report. This will provide a clearly formatted, concrete-mix weight chart for one cubic yard of concrete. 

To use this Python tool, open the ipynb file and follow the directions imbedded in the code. The user will input all the necessary parameters and the code will output the final mix design table.
